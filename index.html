HTML basics

HTML   HyperText Markup Language   the standard markup language for web pages
Basic Structure   <!DOCTYPE> declaration, represents document type
<html> root element of the HTML page
<head> contains meta information about the page
<title> displays text on browser's tab
<body> container for all visible content
HTML Element start tag, content, end tag <tagname> content goes here </tagname>
empty tags tags with no closing tag <br>
Heading Tags <h1><h2><h3><h4><h5><h6> level of importance from left to right
Paragraph Tag <p></p> element containing a block of text
HTML links <a href=""></a> tag to link to another page or section on current page
Images <img src="imagelink"> tag to link an image
alt="description of image" text displayed in case image does not load

CSS basics

CSS   Cascading Style Sheets   language for how the document is presented, or laid out
Basic Structure External Style Sheet <link rel="stylesheet" href="style.css">
Internal Style Sheet <style> content goes here </style>
Inline Styles <p style="color:blue;">
CSS Element element{declarations} p {background: #000;}
CSS Selectors id = #id #id{color: blue;}
class = .class .class{color: red;}
* = all elements * {background: green;}
p = p element p {font-size: 12px;}
CSS Functions rgba(); Red-Green-Blue-Alpha values
hsla(); Hue-Saturation-Lightness-Alpha values
linear-gradient(); creates a linear gradient
CSS Colors color definitions hexadecimal, rgb, hsl, predefined color names
